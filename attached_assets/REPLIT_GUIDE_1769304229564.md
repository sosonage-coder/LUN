# Entity Compliance Calendar - Replit Deployment Guide

## Overview

This is a professional **Entity Compliance Calendar Dashboard** - a management-focused web application designed to showcase governance and compliance capabilities as part of a comprehensive finance tools platform (Netflix of Finance).

The dashboard demonstrates real-time compliance monitoring, risk management, audit readiness, and ROI metrics for enterprise compliance teams.

## Quick Start on Replit

### 1. Import to Replit

1. Go to [Replit.com](https://replit.com)
2. Click "Create" â†’ "Import from GitHub"
3. Paste this repository URL
4. Click "Import"
5. Replit will automatically detect it's a Node.js project

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Run Development Server

```bash
pnpm run dev
```

The application will be available at `https://[your-replit-name].replit.dev`

### 4. Build for Production

```bash
pnpm run build
```

## Technology Stack

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4 + shadcn/ui components
- **Routing**: Wouter (lightweight client-side router)
- **Data Visualization**: Recharts (for future enhancements)
- **Build Tool**: Vite
- **Package Manager**: pnpm

## Customization

### Changing Colors

Edit `client/src/index.css` to modify the color palette:

```css
:root {
  --primary: #1e40af;           /* Deep Blue */
  --chart-1: #3b82f6;           /* Chart colors */
  --chart-2: #10b981;           /* Green */
  --chart-3: #f59e0b;           /* Amber */
  --chart-4: #ef4444;           /* Red */
}
```

### Adding Real Data

Replace the sample data generator with API calls:

```typescript
// In client/src/pages/Home.tsx
const [data, setData] = useState(null);

useEffect(() => {
  // Fetch from your API
  fetch('/api/compliance-data')
    .then(res => res.json())
    .then(setData);
}, []);
```

## Troubleshooting

### Dev Server Won't Start
```bash
# Clear cache and reinstall
rm -rf node_modules .pnpm-store
pnpm install
pnpm run dev
```

### Styling Issues
```bash
# Rebuild Tailwind CSS
pnpm run check
```

### Build Errors
```bash
# Check TypeScript
pnpm run check

# Clear build cache
rm -rf dist
pnpm run build
```
